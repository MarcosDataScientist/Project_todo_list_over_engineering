## Bem-vindo ao Projeto Todo List Over Engineering

Este projeto foi desenvolvido com foco em **Boas Práticas de Programação**, **Arquitetura de Software** e **Engenharia de Dados**. Aqui você encontrará toda a documentação necessária para entender como o sistema funciona, desde o backend em Flask até o frontend em Next.js, além da integração com o Supabase.

## 🚀 Começando

Navegue pelas seções laterais para explorar os diferentes módulos do projeto:

- **[Geral](/docs/category/geral)**: Entenda o fluxo de desenvolvimento, CI/CD e as convenções do projeto.
- **[Backend](/docs/category/backend)**: Detalhes da API Flask, estrutura de pastas e lógica de negócios.
- **[Frontend](/docs/category/frontend)**: Arquitetura do Next.js e consumo da API.
- **[API Reference](/docs/api/todo-list-api)**: Documentação interativa dos endpoints.

## 🛠️ Tecnologias Utilizadas

- **Backend**: Flask + SQLAlchemy (PostgreSQL).
- **Frontend**: Next.js + React.
- **Autenticação**: Supabase Auth.
- **Infra**: Docker + Docker Compose.
- **Documentação**: Docusaurus 3 + OpenAPI Docs.
