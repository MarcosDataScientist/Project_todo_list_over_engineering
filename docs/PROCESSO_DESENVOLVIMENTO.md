# Registro do processo de desenvolvimento

Use este arquivo para documentar decisões, datas e entregas da disciplina de Engenharia de Software.

## Como usar

- Acrescente uma seção por sprint, milestone ou aula, conforme o professor pedir.
- Registre **o quê** foi feito, **por quê** (se relevante) e **como validar** (comandos, prints, links).

## Modelo de entrada

```text
### AAAA-MM-DD — Título curto

- Objetivo:
- Atividades:
- Ferramentas (Supabase / Pytest / Swagger / etc.):
- Próximos passos:
```
